# mlang
## 一个初三党写的**编程语言**！

>特性
- 语法简单
- 有现代的第三方库管理工具(尚未开发)
- 编译器代码简单，可被嵌入
- 解释型语言，即时运行

> 语法示例

部分的语法展示：
```
main {
    print("mlang Test");
    print_twice(1+1);
    result = plus(1, 2);
    print(result);
}

do print_twice(e) {
    repeat 2 {
        print(e);
    }
}

fn plus(a, b) = a + b;
```
